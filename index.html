Selected Research

  • Kidney Cancer 3D CT Segmentation and Prognosis Project (Michigan State Univeristy) – Developed a multi-modal learning pipeline for renal mass prognosis that integrates quantitative imaging and clinical information. The workflow first performs coarse kidney localization using TotalSegmentator, followed by fine-grained tumor and cyst segmentation with nnU-Net. From these refined masks, radiomic features are extracted and fused with patient-level medical data to model prognostic outcomes, enabling more accurate prediction of tumor behavior and treatment response.
  • RESA Project (Zhejiang University Medical Centre) – Identified expressed somatic mutations from single-cell RNA-seq data using the RESA (Recurrently Expressed SNV Analysis) framework to characterize tumor heterogeneity and mutation expression patterns.
